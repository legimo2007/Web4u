1. Propiedad: display: flex

Caso de Uso: Crea un contexto de formato flexible que alinea elementos en una única dimensión (fila o columna), ideal para barras de navegación o alineación simple.

Código de Ejemplo

.contenedor {
    display: flex;
    justify-content: space-between;
}
.item {
    padding: 10px;
}

Especificación Flexbox (W3C)

2. Propiedad: display: grid

Caso de Uso: Permite la creación de diseños complejos en dos dimensiones (filas y columnas), ideal para el layout principal de la página.

Código de Ejemplo

.layout {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* Tres columnas */
    gap: 10px;
}
Resultado en el Navegador
Documentación W3C

Especificación Grid Layout (W3C)

3. Propiedad: box-shadow

Caso de Uso: Aplica una o más sombras alrededor del marco de un elemento, dándole profundidad y resaltando el contenido (efecto 3D o de elevación).

Código de Ejemplo

.tarjeta {
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.3);
}
Resultado en el Navegador
Documentación W3C

Especificación box-shadow (W3C)

4. Propiedad: border-radius

Caso de Uso: Permite redondear las esquinas de un elemento. Es esencial para el diseño moderno de botones, tarjetas e imágenes circulares.

Código de Ejemplo

.boton {
    border-radius: 8px;
}
.circular {
    border-radius: 50%; /* Crea un círculo perfecto */
}
Resultado en el Navegador
Documentación W3C

Especificación border-radius (W3C)

5. Propiedad: transition

Caso de Uso: Permite que los cambios en las propiedades CSS ocurran sin problemas durante un período de tiempo especificado, creando animaciones suaves y atractivas (ej. un cambio de color al pasar el ratón).

Código de Ejemplo

.enlace {
    transition: background-color 0.3s ease-in-out;
}
.enlace:hover {
    background-color: #f0f0f0;
}
